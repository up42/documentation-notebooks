# documentation-notebooks

Jupyter notebooks used in [UP42 Documentation hub](https://docs.up42.com/sdk).

## File linting

This repository uses [pre-commit](https://pre-commit.com/) to lint code blocks in notebooks. Run the following commands to install the pre-commit hooks before submitting changes.

```sh
python -m pip install pre-commit # Install pre-commit
pre-commit install # Install the pre-commit hooks
```
